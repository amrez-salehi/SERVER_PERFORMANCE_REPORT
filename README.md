# SERVER_PERFORMANCE_REPORT
Here’s a simple and portable Bash script (server-stats.sh) that collects and prints all the requested server performance stats — works on most Linux distributions (Ubuntu, Debian, CentOS, Fedora, etc.).

✅ What it shows

CPU usage percentage

Memory total/used/free and usage percentage

Disk usage (total, used, available, percentage)

Top 5 CPU-heavy processes

Top 5 memory-heavy processes

OS info, uptime, load average, logged-in users

Last failed login attempts (if permitted)
